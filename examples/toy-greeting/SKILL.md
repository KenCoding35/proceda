# Toy Greeting Workflow

A simple workflow to demonstrate skill execution with inline widgets.

## Steps

### Step 1: Say Hello

Say "Hello!" to the user in a friendly way.

### Step 2: Ask How They're Doing

Ask the user how they're doing today. Wait for their response.

### Step 3: Corporate HR Reframe

Take the user's reply from Step 2 and reframe it using corporate HR speak. Make it sound like something from a corporate wellness newsletter.

### Step 4: Show Time Remaining

Show the user how much time is left in the day, week, month, and year. Use the available tools to display this information visually.

### Step 5: Nietzsche Overview

[PRE-APPROVAL REQUIRED]

Provide a one sentence overview of Nietzsche's philosophy.
