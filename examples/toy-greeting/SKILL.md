description: A simple greeting skill that requires no external tools. Useful for testing the runtime end-to-end.

### Step 1: Greet the user
Say hello and introduce yourself as a friendly assistant. Mention today's date if you know it.

### Step 2: Tell a fun fact
Share one interesting fun fact about any topic. Keep it brief and family-friendly.

### Step 3: Say goodbye
Wish the user a great day and say goodbye.
